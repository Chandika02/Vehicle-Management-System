# Vehicle-Management-System
Vehicle Service Center Management System using HTML, CSS customer, vehicle, service, status, and payment management.
## Project Overview
This project is developed using HTML, CSS, and JavaScript to manage vehicle service center operations efficiently.

## Features
- Login System
- Dashboard
- Customer Details Management
- Vehicle Details Management
- Service Preference Booking
- Service Status Tracking
- Payment Management

## Technologies Used
- HTML
- CSS
- JavaScript

## Modules
1. Customer Management
2. Vehicle Management
3. Service Booking
4. Service Tracking
5. Payment System

## Future Enhancements
- Admin/User Authentication
- Database Connectivity
- Responsive Design
